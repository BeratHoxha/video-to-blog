# Claude Project Configuration

## Commands

- bootstrap
- create_plan
- implement
- review_branch
- create_pr
- resolve_pr_comments
- improve_prompt

## Workflow

1. Write task in tasks/task.md
2. Run /create_plan
3. Review the generated plan
4. Type YES
5. Run /implement_plan
6. Run /review_branch
7. Run /create_pr

## Principles

- Keep code clean.
- Avoid long lines.
- Small focused tests.
- No large monolithic specs.
- Plan before implementation.
